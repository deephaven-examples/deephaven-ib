deephaven.html¶

This module supports exporting Deephaven data in the HTML format.

to_html(table)[source]¶

Returns a table formatted as an HTML string. Limit use to small tables to avoid running out of memory.

Return type:

str

Returns:

a HTML string

Raises:

DHError –
